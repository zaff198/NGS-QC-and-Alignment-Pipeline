# Dataset

This project uses a teaching dataset containing paired-end reads from *Mycoplasma pneumoniae*.

## Files

- mycoplasma_pneumoniae_F_100K.fastq
- mycoplasma_pneumoniae_R_100K.fastq

Download the dataset from:

**https://doi.org/10.5281/zenodo.21139442**

Extract the files into the `data/` directory before running the pipeline.
